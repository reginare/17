=== HUSBAND BIRTHDAY API v2.1 ===

Добро пожаловать в систему поздравлений. Для доступа к подарку используйте следующие методы.

Base URL: https://[ВашUsername].github.io/husband-birthday-api/

GET /api/memories/first-date

Возвращает данные о первом свидании.

Response: 200 OK (при успехе)

GET /api/reasons/why-i-love-you

Возвращает массив причин в формате JSON. Содержит часть ключа для финального запроса.

Response: 200 OK

GET /api/puzzle/1

Эндпоинт с первой головоломкой. Необходим для получения второй части ключа.

Response: 200 OK

GET /api/puzzle/2

Вторая головоломка. Содержит финальную часть ключа.

Response: 200 OK

POST /api/gift

Метод для получения координат главного подарка. Требует секретный ключ в теле запроса.

Body: {"auth_key": "SECRET_KEY"}

Response: 200 OK (при правильном ключе), 403 Forbidden (при неверном ключе)

Удачи! Система ждет твоих запросов.
